CONCEPTOS

WAMP:

Es el acronimo de Windows, Apache, MySQL y PHP (o Perl o Python) que conforma una plataforma de desarrollo que incluye el sistema operativo (Windows), servidor web (Apache), base de datos (MySQL) y un lenguaje de script (PHP, Perl o Python).

Webmaster:

Es el tecnico responsable de gestionar un sitio web a nivel informatico, y revisa que los contenidos se publiquen correctamente, asi el funcionamiento de la pagina.

Warez:

En el argot de Internet, se refiere a software que haya sido pirateado o copiado ilegalmente.
