CONCEPTOS

Cache:

Un cache es un sistema especial de almacenamiento de alta velocidad. Puede ser tambien un area reservada de la memoria principal como un dispositivo de almacenamiento de alta velocidad independiente.

CD-ROM:

Compact Disc Read Only Memory es un medio de almacenamiento de solo lectura.

Cable UTP:

Cable de par trenzado no apantallado. El conector menos frecuente con el UTP es el RJ45
